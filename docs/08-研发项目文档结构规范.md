## 十、字段数据类型定义

| 类型代码 | 类型名称 | 说明 | 示例值 | 验证规则 |
|----------|----------|------|--------|----------|
| String | 字符串 | 文本字符串 | "智能门锁" | 最大长度 500 字符 |
| Number | 数字 | 整数或小数 | 1000000 | 可带 2 位小数 |
| Date | 日期 | 标准日期格式 | 2025-03-13 | YYYY-MM-DD |
| Select | 单选 | 预定义选项选择 | P0/P1/P2/P3 | 必须从列表选择 |
| User | 用户 | 系统用户引用 | 张三 | 有效系统用户 |
| List | 列表 | 多项数据集合 | [项 1, 项 2, 项 3] | 最多 50 项 |
| Text | 长文本 | 多行文本描述 | ≥100 字的描述 | 最大 10000 字符 |
| Diagram | 图表 | 图形化展示 | 组织架构图 | 支持 PNG/SVG |
| Timeline | 时间线 | 时间轴展示 | 进度计划 | 日期范围 |
| Matrix | 矩阵 | 二维数据表 | 风险评估矩阵 | 行列可定义 |

## 十一、文档间数据流转规则

### 11.1 核心字段流转图

```
┌──────────────┐
│   立项申请    │
└──────┬───────┘
       │ 字段继承
       ↓

└──────┬───────────────────────────────────┘
       │ 字段继承
       ↓
┌─────────────────────────────────────────┐
│            可行性报告                     │
│ ← 项目目标、技术需求、预算、周期、里程碑   │
└──────┬───────────────────────────────────┘
       │ 字段继承
       ↓
┌─────────────────────────────────────────┐
│           正式立项说明                    │
│ ← 项目背景、预算、人力、周期              │
└──────┬───────────────────────────────────┘
       │ 字段继承
       ↓
┌─────────────────────────────────────────┐

       ↓

```

### 11.2 核心字段血缘关系

| 字段 | 源头文档 | 流转路径 | 最终文档 | 同步规则 |
|------|----------|----------|----------|----------|
| 项目名称 | 立项申请 | 所有文档 | 结项报告 | 全程一致，禁止修改 |
| 项目编号 | 立项申请 | 所有文档 | 结项报告 | 全程一致，系统生成 |
| 项目经理 | 立项申请 | 所有文档 | 结项报告 | 变更需审批 |
| 预算总额 | 立项申请 | 所有文档 | 结项报告 | 允许±10% 调整 |
| 开发周期 | 立项申请 | 所有文档 | 结项报告 | 允许±1 月调整 |
| 项目目标 | 立项书 | 章程→结项 | 结项报告 | 变更需走变更流程 |
| 技术目标 | 立项书 | 章程→结项 | 结项报告 | 变更需走变更流程 |
| 风险等级 | 立项申请 | 章程→状态表 | 结项报告 | 可更新 |
| 里程碑 | 立项书 | 章程→状态表 | 结项报告 | 时间只能提前 |

### 11.3 数据一致性规则

| 规则编号 | 规则描述 | 涉及字段 | 校验时机 | 违规处理 |
|----------|----------|----------|----------|----------|
| R1 | 项目名称全程一致 | 项目名称 | 文档创建时 | 阻止创建 |
| R2 | 项目编号全程一致 | 项目编号 | 文档创建时 | 阻止创建 |
| R3 | 预算总额偏差≤10% | 预算总额 | 文档创建时 | 警告 + 审批 |
| R4 | 开发周期偏差≤1 月 | 开发周期 | 文档创建时 | 警告 |
| R5 | 项目经理变更需审批 | 项目经理 | 变更时 | 审批流程 |
| R6 | 里程碑时间只能提前 | 里程碑时间 | 状态更新时 | 阻止延后 |
| R7 | 进度偏差>20% 需升级 | 进度偏差 | 状态更新时 | 自动告警 |
| R8 | 成本偏差>10% 需升级 | 成本偏差 | 状态更新时 | 自动告警 |

### 11.4 字段共享规则

**完全共享字段（所有文档必须一致）:**
- 项目名称
- 项目编号
- 项目类型
- 项目经理

**条件共享字段（允许有限变更）:**
- 预算总额（±10%）
- 开发周期（±1 月）
- 里程碑时间（只能提前）

**独立字段（各文档独立维护）:**
- 技术方案细节
- 风险评估详情
- 团队成员具体分工
