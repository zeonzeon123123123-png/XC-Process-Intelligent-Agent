# 研发工作流程图

## 流程总览

```
产品规划 → 项目立项 → 设计开发 → EVT → DVT → PVT → MP
   │          │          │       │     │     │     │
   ▼          ▼          ▼       ▼     ▼     ▼     ▼
 产品提案   制定计划   结构设计  EVT 试产  DVT 试产  PVT 试产  量产
 需求评审   项目立项   硬件设计  EVT 检验  DVT 验证  PVT 验证  项目结项
 ID 设计    PRD 确认   软件设计  问题优化  问题优化  MP 爬坡
 PRD 编写              开模/手板  EVT 评审  DVT 评审  PVT 评审
```

## 详细流程

### 第一阶段：产品规划

1. 产品提案
2. 需求评审
3. ID 设计
4. PRD 编写
5. PRD 评审
6. PRD 确认

### 第二阶段：项目立项

1. 制定研发计划
2. 项目立项
3. 项目启动
4. PRD 确认

### 第三阶段：设计开发

1. 结构设计
2. 硬件设计（SCH/PCB）
3. 软件设计
4. 开模/手板

### 第四阶段：EVT（工程验证测试）

1. EVT 试产
2. EVT 检验
3. 问题优化
4. EVT 评审

### 第五阶段：DVT（设计验证测试）

1. DVT 试产
2. DVT 验证
3. 问题优化
4. DVT 评审

### 第六阶段：PVT（生产验证测试）/ MP（量产）

1. PVT 试产
2. PVT 验证
3. MP 爬坡
4. 项目结项

---

**详细流程说明请参考各部门工作方案文档。**
