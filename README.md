## ✨ 系统功能

### 🔐 登录注册
- 用户名密码登录
- 完整注册流程（用户名、密码、部门、职务、邮箱、手机）
- 管理员默认账号：`admin` / `Admin@2026`

### 🏠 首页
- **5 个部门智能体入口**：产品部、项目部、硬件部、软件部、结构部
- **项目触发**：输入 PRD 或上传项目提案创建新项目
- **我的项目**：查看所有项目及当前阶段
- **项目流转可视化**：立项 → 规划 → 设计 → EVT → DVT → PVT → MP

### 📊 项目部智能体（核心）
- **项目部文档入口**：8 份项目文件
- **文档模板**：
  - 立项书（18 个字段）
  - 可行性报告（15 个字段）
  - 正式立项说明（8 个字段）
  - 项目章程（16 个字段）
  - 项目状态表（16 个字段）
  - 结项报告（13 个字段）
  - 成果报告（11 个字段）
  - 评审流程（12 个字段）
- **文档自动生成**：填写字段后自动保存
- **项目阶段追踪**：实时显示当前阶段

### 🔧 管理后台（完整权限）
管理员拥有所有增删改权限：
- **数据概览**：用户数、项目数、文档数统计
- **用户管理**：添加、编辑、删除用户
- **文档模板管理**：管理文档模板
- **字段管理**：增删改字段定义
- **流转规则管理**：配置项目阶段流转规则
- **同步规则管理**：配置文档间字段同步规则
- **部门管理**：管理部门架构

## 🚀 使用指南

### 1. 登录系统
- 打开上面的链接
- 使用管理员账号登录：`admin` / `Admin@2026`
- 或注册新用户

### 2. 创建项目（项目触发）
- 在首页填写项目名称、类型
- 输入项目描述/PRD 内容
- 上传文档（PRD、项目提案等）
- 选择负责部门
- 点击"创建项目"

### 3. 使用项目部智能体
- 点击"项目部智能体"入口
- 查看项目当前阶段（流程可视化）
- 选择文档类型（8 份项目文件）
- 填写表单字段
- 点击"生成文档"保存

### 4. 管理后台
- 点击"管理后台"
- 管理用户、文档模板、字段、流转规则、同步规则等

## 📊 项目流转流程

```
立项 → 规划 → 设计 → EVT → DVT → PVT → MP
 ↓      ↓      ↓     ↓     ↓     ↓     ↓
立项书  项目章程 设计文档 EVT 报告 DVT 报告 PVT 报告 结项报告
可行性报告        状态表
正式立项说明
```

### 流转规则

| 转换 | 条件 |
|------|------|
| 立项 → 规划 | 立项书已审批 |
| 规划 → 设计 | 项目章程已确认 |
| 设计 → EVT | 设计评审通过 |
| EVT → DVT | EVT 评审通过，无 P0/P1 问题 |
| DVT → PVT | DVT 评审通过，模具验收完成 |
| PVT → MP | PVT 评审通过，直通率达标 |

## 📄 文档模板字段示例

### 立项书（18 个字段）
- 项目名称、项目编号、产品型号
- 项目类型（自研/JDM/ODM/OEM）
- 优先级（P0/P1/P2/P3）
- 风险等级（高/中/低）
- 技术可行性、市场可行性、资源可行性
- 技术目标、市场目标、商业目标
- 人力需求、预算需求、开发周期
- 关键里程碑、技术风险、应对措施

### 可行性报告（15 个字段）
- 项目名称、编写目的、范围
- 技术可行性评估结论
- 技术风险评估
- 公司现有技术状况
- 关键产品需求分析
- 整体技术方案、硬件/软件/结构技术方案
- 关键技术分析
- 人力资源、设备资源、预算需求
