# XC-Process-Intelligent-Agent

研发工作流程智能体配置中心

## 🌐 在线访问

**智能体配置网页：** [index.html](https://github.com/zeonzeon123123123-png/XC-Process-Intelligent-Agent/blob/main/index.html)

> 下载 `index.html` 文件后在浏览器中打开即可使用

## ✨ 功能特性

### 🔐 用户系统
- **登录注册**：支持用户注册和登录
- **权限管理**：管理员拥有所有权限，普通用户拥有个人权限
- **密码安全**：任何用户密码不能与管理员密码相同

### 📇 企业通讯录
- **组织架构**：按部门展示公司架构
- **员工信息**：显示姓名、职务、部门、在线状态
- **类似飞书**：清晰的层级结构，方便查找联系人

### 👤 个人主页
- **个人信息**：查看自己的基本信息（部门、职务、邮箱、手机）
- **使用统计**：参与项目、完成任务、工作时长、登录天数
- **密码管理**：支持修改个人密码

### 🔧 管理员后台
- **用户管理**：查看、编辑、删除所有用户
- **部门管理**：查看各部门人员分布
- **数据统计**：总用户数、今日登录、活跃用户、新增用户

### 📄 文档中心
- 各部门工作方案文档
- 智能体搭建方案
- 研发项目文档结构规范

## 🚀 快速开始

### 使用智能体配置网页

1. **下载文件**
   - 点击 `index.html` 文件
   - 点击右上角 "Raw" 按钮
   - 右键保存为 `index.html`

2. **打开使用**
   - 在浏览器中打开 `index.html`
   - 默认管理员账号：
     - 用户名：`admin`
     - 密码：`Admin@2026`

3. **注册用户**
   - 点击 "立即注册"
   - 填写用户名、密码、部门、职务等信息
   - 注册完成后自动登录

4. **查看通讯录**
   - 登录后点击导航栏 "通讯录"
   - 按部门查看所有员工信息

5. **个人主页**
   - 点击 "个人主页" 查看个人信息和使用统计
   - 支持修改密码

6. **管理后台**（仅管理员）
   - 管理员登录后点击 "管理后台"
   - 管理所有用户和部门
