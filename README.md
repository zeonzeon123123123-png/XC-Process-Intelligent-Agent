## ✨ 功能特性

### 🔐 用户系统
- **登录注册**：支持用户注册和登录
- **权限管理**：管理员拥有所有权限，普通用户拥有个人权限
- **密码安全**：任何用户密码不能与管理员密码相同

### 📇 企业通讯录
- **组织架构**：按部门展示公司架构
- **员工信息**：显示姓名、职务、部门、在线状态
- **类似飞书**：清晰的层级结构，方便查找联系人

### 👤 个人主页
- **个人信息**：查看自己的基本信息（部门、职务、邮箱、手机）
- **使用统计**：参与项目、完成任务、工作时长、登录天数
- **密码管理**：支持修改个人密码

### 📊 文档分析
- **12 种文档类型**：基于 24 个项目、160 份文档分析
- **核心文档**：立项书、可行性报告、正式立项说明、项目章程
- **研发流程**：可视化展示标准研发流程

### 🔧 管理员后台
- **用户管理**：查看、编辑、删除所有用户
- **部门管理**：查看各部门人员分布
- **数据统计**：总用户数、今日登录、活跃用户、新增用户

### 📄 文档中心
- 各部门工作方案文档
- 智能体搭建方案
- 研发项目文档结构规范

## 🚀 快速开始

### 使用智能体配置网页

1. **直接打开网页**
   - 点击上面的 "点击这里打开智能体配置网页" 链接
   - 或者访问：https://raw.githack.com/zeonzeon123123123-png/XC-Process-Intelligent-Agent/main/index.html

2. **登录系统**
   - 使用管理员账号登录：
     - 用户名：`admin`
     - 密码：`Admin@2026`

3. **注册新用户**
   - 点击 "立即注册"
   - 填写用户名、密码、部门、职务等信息
   - 注册完成后自动登录

4. **使用功能**
   - **通讯录**：查看公司组织架构和员工信息
   - **个人主页**：查看个人信息和使用统计
   - **文档分析**：查看研发项目文档分析数据
   - **管理后台**：管理员登录后查看（仅管理员可见）
