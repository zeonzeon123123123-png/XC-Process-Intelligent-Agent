📁 文件管理系统

智能体自动管理项目文件，支持自动编号、版本控制、回收站功能

📋 文件管理系统配置

  • 智能体自动编号：根据工作流程顺序自动分配文件夹编号（00-项目公共，01-产品部，02-项目部...）
  • 文件命名规范：{文档类型}_{版本号}_{创建者}_{日期}，智能体自动补全
  • 版本管理：文件夹只显示最新版本，历史版本可通过 Git history 查询
  • 回收站机制：删除的文件保留 30 天，支持找回，无需审批
  • 部门顺序：产品部 → 项目部 → 硬件部 → 结构部 → 软件部 → 测试部
